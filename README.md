An AI chatbot built with Python and Ollama using the Mistral model.

## Features
* Fast AI responses
* No API key required
* Runs completely on your computer
* Terminal-based chatbot
* Uses the Mistral model with Ollama

## Technologies Used
* Python
* Ollama
* Mistral

## How to Run
1. Install Ollama.
2. Pull the model:
   ollama pull mistral
3. Install Python package:
   pip install ollama
4. Run the chatbot:
   python bot.py
